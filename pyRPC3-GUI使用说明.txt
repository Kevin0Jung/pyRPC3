pyRPC3-GUI 使用说明
====================

适用对象
--------
这个程序用于在 Windows 电脑上进行 RSP 和 CSV 的双向转换。
对方电脑不需要安装 Python，也不需要配置开发环境。


启动程序
--------
1. 双击运行 dist\pyRPC3-GUI.exe。
2. 程序会在本机启动一个本地服务，并自动打开浏览器页面。
3. 如果浏览器没有自动打开，可以手动访问：

   http://127.0.0.1:8765

注意：程序运行期间不要关闭 pyRPC3-GUI.exe 进程，否则浏览器页面将无法继续转换文件。


功能一：RSP 转 CSV + JSON
------------------------
1. 在左侧区域「RSP -> CSV + JSON」中，拖拽一个 .rsp 文件到上传框。
   也可以点击上传框后选择文件。
2. 点击「转换」按钮。
3. 转换完成后页面会显示两个下载文件：

   - .csv 文件
   - .rpc3meta.json 文件

这两个文件是一组。后续如果要把 CSV 转回 RSP，需要同时保留并使用这个 JSON 文件。


功能二：CSV + JSON 转 RSP
------------------------
1. 在右侧区域「CSV + JSON -> RSP」中，上传 .csv 文件。
2. 再上传与该 CSV 配套的 .rpc3meta.json 文件。
3. 点击「转换」按钮。
4. 转换完成后页面会显示生成的 .rsp 文件下载链接。

重要：CSV 转 RSP 时必须提供配套的 .rpc3meta.json 文件。
这个 JSON 文件是在 RSP 转 CSV 阶段自动生成的，用来保存原始 RSP 的头信息、通道信息和 scale 信息。
没有这个 JSON 文件，生成的 RSP 可能无法保持和原文件完全一致。


输出文件位置
------------
浏览器下载按钮会直接下载转换结果。

程序内部的临时文件默认保存在当前 Windows 用户目录下：

%LOCALAPPDATA%\pyRPC3\output\gui

一般用户不需要手动进入这个目录。


支持的文件
----------
RSP 转 CSV：
- .rsp
- .drv
- .rpc
- .tim

CSV 转 RSP：
- .csv
- .rpc3meta.json


常见问题
--------
1. 浏览器打不开页面
   - 确认 pyRPC3-GUI.exe 仍在运行。
   - 手动打开 http://127.0.0.1:8765。

2. CSV 转 RSP 提示需要 JSON
   - 请上传 RSP 转 CSV 时生成的 .rpc3meta.json 文件。
   - JSON 文件要和当前 CSV 属于同一次转换结果。

3. 文件很大时转换时间较长
   - 请等待页面从「处理中...」变为「完成」。
   - 转换过程中不要刷新页面或关闭程序。

4. Windows 安全提示
   - 这是本地打包程序常见提示。
   - 程序只在本机 127.0.0.1 启动本地页面，不需要访问外网。


一句话流程
----------
RSP 转 CSV：上传 .rsp -> 下载 .csv 和 .rpc3meta.json。

CSV 转 RSP：上传 .csv 和对应 .rpc3meta.json -> 下载 .rsp。
